🚀 Installation Steps

Step 1: Download the manifest-working.xml file (🔥 RECOMMENDED - Microsoft pattern)
Step 2: Go to https://aka.ms/olksideload
Step 3: Sign in with your Microsoft account
Step 4: Click "Upload manifest file" and select the downloaded XML file
Step 5: Wait for installation to complete
Step 6: Open Outlook Web, compose a new email, and look for "Project Email" group in the ribbon
